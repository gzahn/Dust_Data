# Dust_Data

Code to process and analyze Chaudhary Lab dust and soil data from NEON sites.


Full pipeline can be run via:

```./pipeline.sh```


Tested on Ubuntu 22.04 and Red Hat 8.10 Linux systems using R v 4.4.0
